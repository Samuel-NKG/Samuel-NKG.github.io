# 这是什么？
这是一个基于Hexo的博客主题，采用Material Design风格，当然也融合了一部分md3e的设计，基于MDUI框架，同时也支持响应式设计，在移动端也有很好的显示效果。
# 特性
- 基于Material Design 3 设计风格
- 注释很多
- 适配多种语言
- 响应式设计，适配移动端
